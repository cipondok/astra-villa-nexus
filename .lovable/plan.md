

# Redesign Verified Badge to Circular Seal Style

## Overview

Replace the current **shield-shaped** verified badge with a modern **circular starburst/seal** style -- a round badge with wavy/scalloped edges, the user's own logo centered inside in white, and a small checkmark overlay. This is a clean, flat, professional look commonly seen on travel and social media platforms.

## Visual Design

The new badge will have:
- **Circular base** with wavy/scalloped outer edge (12-16 points) creating a seal/rosette shape
- **Gradient fill** using the level colors (light to dark)
- **White logo** centered inside (the user's branding logo, rendered with a white filter)
- **Small checkmark circle** in the bottom-right corner
- **Clean flat design** with subtle drop shadow
- No shield, no facets -- just the clean seal shape

## Technical Details

### Starburst path generation

The wavy edge is generated by alternating between an outer radius and inner radius at evenly spaced angles:

```text
For N points (e.g., 14):
  angle = i * (2*PI / N)
  radius = (i % 2 === 0) ? outerR : innerR
  x = cx + radius * cos(angle)
  y = cy + radius * sin(angle)
```

This creates the characteristic seal/rosette shape.

### Backward compatibility

The `shieldStyle` field in saved settings will be mapped:
- `"diamond"` maps to starburst seal
- `"classic"` maps to scalloped seal  
- `"minimal"` maps to clean circle

New saves will use the new field name `badgeShape` with values `"seal"`, `"scallop"`, `"circle"`.
